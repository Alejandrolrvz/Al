Cybersecurity Portfolio


HomeLab

Azure Active Directory

Leveraged Azure Active Directory to establish security groups, accurately assigning users and adhering to the least privilege principle by carefully selecting privileges. Implemented comprehensive access management strategies to fortify data security, enabling seamless collaboration while preserving system integrity.

Virtual Machine

Deployed two Windows virtual machines. The first VM served as a target with disabled security features and enabled logging. The second VM acted as an attack machine, generating noise and capturing logs for analysis. Additionally, the vulnerable VM was exposed on the internet as a honeypot to attract potential attackers.

Logging & Monitoring

Conducted meticulous analysis of logs generated by attempted attacks on the virtual machines. Leveraged the power of KQL (Kusto Query Language) to perform in-depth log analysis, identifying patterns, anomalies, and potential security threats. This proactive approach enhanced our ability to detect and mitigate risks

Key Vault

Established a centralized Key Vault for relevant resource groups, ensuring secure storage and management of sensitive credentials. Implemented fine-grained access controls, granting "contributor" access only to specific users requiring it. This safeguarded critical data and minimized unauthorized access risks within our homeland's infrastructure.

Microsoft Sentinel

Utilized Microsoft Sentinel as our primary incident response platform, enabling rapid detection, investigation, and response to security incidents and attacks. Leveraged advanced threat intelligence, automation, and real-time monitoring to effectively mitigate risks. Additionally, implemented data ingestion and created a geographic map to visualize and plot the attacks geographically, enhancing situational awareness and strategic decision-making.

Network Security Group and Firewall

Established a secure Demilitarized Zone (DMZ) by creating a network security group and implementing a robust firewall. Effectively segmented and protected our network infrastructure, allowing secure communication with external entities while maintaining strict access controls and preventing unauthorized access to critical internal resources within our homeland.



Offensive Projects

Banner Grabber

I developed a Banner Grabber that identifies unsecured ports and applications, enabling targeted attacks based on vulnerabilities. This project enhanced my skills by deepening my understanding of the adversary mindset, making me a better defender. It provided valuable insights into identifying weaknesses and fortified my ability to implement proactive security measures.

Brute force

I developed a brute force project where I gained insights into weak passwords' vulnerabilities and the importance of strong authentication. This knowledge aids in designing robust security measures to protect systems from such attacks. Additionally, I learned countermeasures like account lockouts, rate limiting, and encryption techniques to secure sensitive data.

IP Lookup

I developed an IP Lookup script that enhances my skills as a cybersecurity analyst by providing a quick and efficient method to obtain the IP address of a website. This tool aids in network analysis, threat hunting, and identifying potential malicious activities, ultimately strengthening my ability to safeguard against cyber threats.

Key Logger

I developed a keylogger in Python that captures and logs user keystrokes on a computer, saving them to a specified file location. This script provides valuable insights into keylogging techniques and their potential implications, enhancing Python programming skills and promoting a deeper understanding of computer security.

DDoS Attack

I developed a WiFi DDoS Python Script that offers insights into the adversarial mindset while honing Python programming skills. This project explores WiFi discovery techniques and simulates DDoS attacks on identified networks, providing valuable experience in understanding network vulnerabilities and the importance of robust security measures.

Number Tracker

I developed a number tracker Python project that enhances my skills as a cybersecurity analyst by providing insights into geolocation and phone number tracking. By correlating phone numbers with city and state information, this project aids in investigations, threat intelligence, and identifying potential risks related to phone-based attacks.



Defense Projects

vulnerability Scanner

I developed a Network Vulnerability Scanner that utilizes Nmap to identify vulnerabilities in targeted networks. With automated network discovery, vulnerability detection, and detailed reporting, this project enhances my security analyst skills and provides hands-on experience in assessing network security.

Port Scanner

I developed a port scanner that enhanced my analyst skills by identifying vulnerable ports for secure configuration. I can swiftly assess vulnerabilities, strengthen target system security, and mitigate risks. This project is a valuable tool for network security assessment, advancing expertise in vulnerability detection and remediation.
